## 📋 Sobre o Projeto

Portfólio profissional desenvolvido com **HTML5**, **CSS3** e **JavaScript Vanilla** (sem frameworks), demonstrando domínio de tecnologias web fundamentais.

### ✨ Funcionalidades

- **Estrutura Semântica HTML5** — Uso correto de tags semânticas (`header`, `section`, `footer`, `nav`, `article`)
- **Layout com Flexbox e CSS Grid** — Design responsivo e organizado
- **Design Responsivo (Mobile-First)** — Funciona perfeitamente em todos os dispositivos
- **Renderização Dinâmica com JavaScript** — Projetos carregados dinamicamente a partir de um array de objetos
- **Animações CSS3** — Hover effects, scroll animations e transições suaves
- **Efeito de Digitação (Typing)** — Texto animado no hero section
- **Menu Responsivo** — Hamburger menu para dispositivos mobile
- **Intersection Observer API** — Animações ativadas ao scroll
- **Acessibilidade** — ARIA labels, contraste adequado e suporte a `prefers-reduced-motion`

## 📱 Responsividade

O portfólio foi desenvolvido com abordagem **Mobile-First** e se adapta a:

- 📱 **Mobile** (< 768px) — Menu hambúrguer, layout em coluna
- 📱 **Tablet** (768px - 1023px) — Grid de 2 colunas para projetos
- 💻 **Desktop** (≥ 1024px) — Layout completo com todas as funcionalidades
